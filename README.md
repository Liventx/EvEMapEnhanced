# EvE Map Enhanced

Десктопная карта вселенной EVE Online с планировщиком маршрутов, расчётом дальности прыжка и
отслеживанием персонажей в реальном времени. Интерфейс на русском языке.

**Текущая версия:** [Alfa 1.4](https://github.com/Liventx/EvEMapEnhanced/releases/tag/alfa-1.4)

## Что умеет

### Карта

- Схематическая карта (в стиле Dotlan) и карта в реальных координатах — переключаются на лету
- Системы показываются плашками: имя + счётчик NPC-киллов вблизи, только имя чуть дальше и точки на общем обзоре
- Раскраска систем по активности **NPC Kills** или по **статусу безопасности**
- Названия регионов на обзоре вселенной, названия систем — при приближении
- Подсказка с названием системы при наведении, когда карта отдалена
- Системы с NPC-станцией помечаются небольшим салатовым квадратом
- Плавное масштабирование колёсиком мыши и слайдером

### Маршруты и прыжки

- Построение маршрута **ОТ → ДО** через гейты, прыжки и jump bridges
- Подсветка дальности прыжка от выбранной системы (учитывает корабль, навыки и метод прыжка)
- Симуляция дальности прыжка с несколькими точками и подсветкой пересечения
- Отдельная мини-карта дальности прыжка в реальном масштабе для планирования цепочек Black Ops
- Расчёт jump fatigue и топлива
- Сохранение маршрутов и заметок к системам

### Отслеживание в реальном времени

- Вход через аккаунт EVE Online (ESI SSO)
- Маяк «вы здесь» на карте для основного персонажа
- Отдельные маяки для профилей **Cyno** (голубой) и **SC** (тёмно-синий)
- Режим **Focus** — фиксирует точку отсчёта дальности прыжка при кликах по карте

### Активность и структуры

- Подсветка PvP-активности по данным zKillboard (горячие / недавние / NPC-капиталы)
- Свои структуры (Fortizar, Keepstar, Ansiblex, cyno-маяки и другие) на карте
- Открытие страницы системы на zKillboard из контекстного меню

## Быстрый старт

1. Скачайте и установите последнюю версию.
2. Запустите приложение.
3. Меню **Данные → Скачать / обновить SDE** — один раз загрузите игровые данные (нужен интернет).
4. *(По желанию)* **Аккаунт → Войти в EVE Online** — чтобы видеть навыки, местоположение и NPC-киллы.
5. Кликните систему на карте, задайте **ОТ** и **ДО**, нажмите **Построить маршрут**.

Все данные хранятся локально на вашем компьютере.

## Что нового в Alfa 1.4

- Подсказка с названием системы при наведении, когда карта отдалена и видны только точки
- Плашки систем стали компактнее — карта читается плотнее и аккуратнее
- Системы с NPC-станцией отмечаются салатовым квадратом в углу плашки
- Названия регионов дольше остаются поверх карты при приближении
- Плашки «только имя» используются на более широком диапазоне масштаба

### Alfa 1.3

- Слайдер масштаба на верхней панели с числовым значением, синхронизирован с колёсиком мыши
- Единый вид плашек во всех регионах на одном масштабе
- Маяки персонажей аккуратно уходят под плашки при приближении

### Alfa 1.2

- Симуляция дальности прыжка с несколькими точками и подсветкой пересечения
- Кнопка центрирования карты на основном персонаже
- Повторный запуск выводит уже открытое окно вместо создания второго

### Alfa 1.1

- Названия регионов поверх карты на широком масштабе
- Маяки и подсветки уменьшаются при сильном отдалении
